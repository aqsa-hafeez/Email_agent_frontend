# 🖥️ Smart AI Email Dashboard - Frontend

This is the user interface for the AI Email Agent, built with **Next.js 14** and **Tailwind CSS**. It provides a real-time view of your inbox and allows you to approve AI-generated replies.

## ✨ Features
- **Live Sync:** Automatically refreshes the inbox every 10-15 seconds.
- **AI Draft Preview:** View and review drafts generated by the LLaMA-3 model.
- **One-Click Approval:** Send professional replies directly from the dashboard.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
- **Status Indicators:** Clear badges for "Delivered", "Pending", and "Syncing" states.

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **API Client:** Axios
